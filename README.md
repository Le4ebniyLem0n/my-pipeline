# My CI/CD Pipeline

Автоматическая проверка качества кода и уведомления в Telegram при мерже Pull Request.

## Технологии

- **GitHub Actions** — CI/CD платформа
- **Python 3.11** — язык для проверки кода
- **Flake8** — линтер для Python
- **Telegram Bot API** — отправка уведомлений

## Требования

- Git установленный локально
- Telegram бот (для уведомлений)
